# DocuMind AI

AI-powered RAG-based PDF Question Answering System using LangChain, ChromaDB, HuggingFace Embeddings, and Gemini API.

## Features
- Upload PDFs
- Semantic search
- AI-powered question answering
- Recommendation system
- Streamlit UI

## Tech Stack
- Python
- Streamlit
- LangChain
- ChromaDB
- Gemini API
- Sentence Transformers

## Setup
pip install -r requirements.txt

streamlit run frontend/streamlit_app.py
